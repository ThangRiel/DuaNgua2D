# 🏇 Horse Racing Simulator

## 📖 Giới thiệu
Đây là một game mô phỏng đua ngựa đơn giản, đóng vai trò là một module trong một chuỗi dự án quản lý lớn hơn. Chức năng chính của game là mô phỏng một cuộc đua trực quan và tự động gửi API vị trí, cũng như thứ tự về đích của các chú ngựa lên trang web quản lý hệ thống.

## 🎮 Gameplay
Lối chơi được thiết kế tối giản nhất có thể để tập trung vào việc test luồng dữ liệu:
1. Mở game.
2. Bấm nút **Chạy**.
3. Xem đồ họa mô phỏng ngựa chạy trên đường đua.
4. Nhận kết quả bảng xếp hạng thứ tự ngựa khi kết thúc.

## 📥 Cách tải và chạy game
1. Chuyển sang mục **[Releases](https://github.com/ThangRiel/DuaNgua2D/releases)** ở cột bên phải.
2. Tải file `.zip` của phiên bản mới nhất về máy.
3. Giải nén toàn bộ thư mục.
4. Chạy file `.exe` bên trong để bắt đầu. *(Lưu ý: Không kéo file .exe ra khỏi thư mục giải nén)*

## ⚙️ Kỹ thuật
* **Platform:** Windows
* **Engine:** Unity
* **Tích hợp:** Gửi request API (vị trí, kết quả chung cuộc) trực tiếp đến backend quản lý.
